# 采坑记

  1. 123.toString 报错
js中 integer 默认是双精度浮点数，这里小数点会被当做浮点数看待

(123).toString()
123..toString()

toString 默认传递10，表示转换为10进制

const presets = [
	[
		"@babel/env",
		{
			targets: {
				edge: "17",
				chrome: "64",
				firefox: "67",
				safari: '11.1'
			},

            //Babel将检查你的所有代码, 然后查找目标环境中缺少的功能, 并引入仅包含所需的polyfill

	]
